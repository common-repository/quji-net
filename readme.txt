== Description ==

趣迹是一个直观了解用户行为的可视化用户行为分析工具,趣记会为你记录用户的操作轨迹,并利用大数据分析用户的点击、移动和滚动行为生成用户的行为操作热力图。你可以从定性和定量双角度分析用户的操作行为，改善产品的体验，提高转化率。

* **用户行为轨迹记录** - 回放用户在你网页上所做的一切消除猜测，你可以看到用户在你网页上的每一次鼠标移动，滚动，点击等操作，帮助你直观了解用户使用产品时遇到的问题，定性分析用户使用产品的行为轨迹，运用同理心知道用户的心理模型改善产品的用户体验，提高转化率，观其踪，知所想，人人都是用户体验师。

* **热力图** - 趣记会为你统计在特定时间段内电脑、手机和pad端用户的点击、移动和滚动行为，通过热力图的表现形式直观的展示用户的操作热区，了解用户想要在你的网站上做什么和用户真正关心的是什么，优化页面布局，提高产品体验。通过热力图了解新功能或新改版的效果如何，是否和预期相同，帮助你发现问题，定位问题。

= 如何使用趣迹 =

1.注册趣迹,填写您的网站地址。

2.获得安装代码，复制粘贴到您的网站代码的Head中。

3.安装成功后访问quji.net您可以看到您的用户的操作轨迹。
